##System: All

##Requirements:
* SSH environment.

##Quick Start
1. Set up SSH
	*On Windows, install Git bash
	*On Linux, install OpenSSH


2. Copy Configures:

	Copy `config` to `~/.ssh/config`

3. Additions:

	Use aliases to accelerate command typing. For example:
		alias remote='ssh _hostID_'
	Suppose _hostID_ has been setup in config file.
